## 📌 About The Project

**Siri-Dhanya Hub** is an innovative Android application developed to promote **millet awareness**, **healthy food habits**, and **sustainable agriculture** using modern Android Development and GenAI technologies.

The application helps users explore different millet varieties, understand their nutritional benefits, and generate AI-powered recipe suggestions in a simple and interactive way.

---

## ✨ Key Features

🌾 **Millet Information Library**  
Explore detailed information about:
- Ragi
- Navane
- Sajje
- Baragu
- Foxtail Millet
- Little Millet and more

🥗 **Nutritional Values**  
View:
- Protein
- Fiber
- Minerals
- Calories
- Vitamins
- Health Benefits

🤖 **AI Recipe Generator**  
Generate smart millet-based recipes using GenAI APIs.

🔍 **Smart Search Functionality**  
Search millet details instantly.

🌐 **Multi-language Support**  
Supports:
- English
- Kannada

🎤 **Voice Assistant Support**  
Easy navigation using voice guidance.

📱 **Modern Android UI**  
Clean, responsive, and user-friendly interface.

# 🎯 Problem Statement

Many millet farmers lack proper digital visibility and consumers have limited knowledge about millet nutrition and recipes.

Siri-Dhanya Hub solves this problem by creating a digital ecosystem that:
- Promotes millet awareness
- Encourages healthy food habits
- Supports sustainable agriculture
- Helps younger generations learn about traditional grains
